Tools Required

  • Plastic Prying Tool
  • M3 hex Key
  • 3/32" hex Key
  • 5/64" hex Key

Parts Included

  • 1 x Stainless Steel Shift Knob Core
  • 1 x Acetal Outer Sleeve
  • 1 x Shift Knob Cap
  • 1 x 3/8" 10-32 Stainless Steel Set Screw
  • 6 x 9/16" 2-56 Stainless Steel Socket Head Cap Screws

Note: This product is designed to fit the 2021+ Ford Bronco with a manual transmission. It will not fit the automatic transmission models. Please ensure you have the correct model before proceeding with the installation.

Procedure

  1. Use a plastic prying tool to pry up and remove the OEM center cap.
  2. Use an M3 hex key to remove the screw in the center of the shift knob.
  3. Remove the shift knob by pulling up.
  4. Install the stainless steel shift knob core onto the shifter.
  5. Align the hole on the side of the core with the flat section of the shifter shaft.
  6. Use an 3/32" hex key to install the stainless steel set screw into the core and tighten.
  7. Re-install the screw removed in step two to secure the core to the shaft.
  8. Slide the black acetal outer sleeve onto the core, keeping the holes aligned. The rotation of the sleeve does not matter, as long as the holes are aligned with the holes in the core.
  9. Place the cap onto the sleeve, keeping the holes in all three parts aligned, then secure it all together with the six stainless steel screws using the 5/64" hex key.
